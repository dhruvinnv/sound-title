Sound Title Privacy Policy

Effective Date: February 2026
Last Updated: February 8, 2026

Overview

Sound Title is a browser extension that enables copyright registration search via the Copyright.gov Public Records API. This privacy policy explains how we collect, use, and protect your information.

Information We Collect

Data Stored Locally

Sound Title stores the following data locally on your device using Chrome's storage API:

  • Search history: ISRC codes, track names, artist names, and search results
  • User preferences: Extension settings and UI state

Important: This data never leaves your device and is not transmitted to any servers operated by Sound Title or Third Chair.

Data Transmitted to Third Parties

Sound Title makes API requests to:

  1. Copyright.gov Public Records API (https://api.publicrecords.copyright.gov)
    • Purpose: To search copyright registrations
    • Data sent: ISRC codes, track names, artist names (as provided by you in search queries)
    • Data received: Copyright registration records from public database
    • Privacy: See Copyright.gov's privacy policy at https://www.copyright.gov/privacy/

External Communication

Sound Title can receive messages from authorized domains:

  • https://usethirdchair.com - Our web application that integrates with this extension

Messages contain search parameters (ISRC, track name, artist names) necessary to perform copyright searches. No personal identification data is transmitted.

How We Use Your Information

  • Search history is stored locally to provide quick access to previous searches in the extension side panel
  • No analytics or tracking data is collected
  • No user identification data is collected or stored
  • No data is sold or shared with third parties (except Copyright.gov API queries)
  • No cookies are set or used

Data Retention

  • All data is stored locally on your device in Chrome's local storage
  • You can clear search history at any time through the extension's side panel interface (History tab → Clear History button)
  • Uninstalling the extension completely removes all stored data from your device

Your Rights

You have the right to:

  • View your search history at any time (Extension side panel → History tab)
  • Clear your search history at any time (Clear History button)
  • Disable the extension temporarily
  • Uninstall the extension and delete all associated data

Data Security

  • All data is stored locally using Chrome's secure storage APIs
  • API communication with Copyright.gov uses HTTPS encryption
  • Extension uses Chrome's Manifest V3 with strict Content Security Policy

Children's Privacy

Sound Title does not knowingly collect information from children under 13. The extension is designed for professional use by music industry professionals and rights holders.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected with a new "Last Updated" date. Continued use of the extension after changes constitutes acceptance of the updated policy.

Contact Us

For questions about this privacy policy or Sound Title:

  • Website: https://usethirdchair.com

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • Chrome Web Store's Limited Use Policy
  • General Data Protection Regulation (GDPR) principles
  • California Consumer Privacy Act (CCPA) principles

User data is used solely to provide the copyright search functionality and is not used for any other purpose.
